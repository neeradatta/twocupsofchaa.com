Customization Services

We can customize these files for you

You asked and we deliver! This option is being made available to our customers. If you purchase the customize listing alongside your purchase of a digital template we will customize the file for you and email you a finalized file.

:: How does this work

1. Add a file you would like us to edit for you from the Two Cups Of Chaa store
2. Add the customize my listing
3. Send details - we will be in email contact with you so please provide a valid email for us to work with
4. We will schedule two half hour sessions for us to work though the wording in the file you would like. This can be done via email or zoom.
5. We will delived the finalized product ready for you to download or print.

:: How much does this cost? The cost of the file you would like us to edit or customize will vary. The cost for custom edits are broken down into three categories:

1. File review and minimal changes (up to 5 edits) $25
2. File review with two phone or zoom sessions (up to half hour) plus delivery of a fully customized digital file which you may then print yourself or give to a printing company for final print and delivery $75
3. Full Custom Package. This package will be used for events that want to feature the photographs of the Bride and Groom outfit embroidery. This option is similar to option 2 but will include photographs of your outfit to carry that theme throughout the entire event. Please contact us for more details. $100
